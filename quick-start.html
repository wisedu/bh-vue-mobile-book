快速上手

快速上手

项目依赖

  1. Node@>=4.0.0
  2. npm(一般不用单独安装, Node 自带)
  3. vue-cli

安装 vue-cli 

$ npm i vue-cli -g

如果速度慢的话使用国内镜像

$ npm i cooking -g --registry=https://registry.npm.taobao.org

创建第一个项目

$ vue init wisedu/bh-mobile-template ${your_project}

启动项目

项目启动后会自动开启监听模式和 HMR, 实时更新界面

$ npm install
$ npm run dev
