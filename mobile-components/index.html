移动端框架简介

移动端组件

介绍

移动端组件基于一个轻量级的开源 ui 库 mint-ui 开发 fork 了一份 bh-mint-ui

安装

$ npm install bh-mint-ui

使用

在脚手架环境下, 直接引用相应组件, 不需要显示的引入样式文件, 推荐使用局部注册的形式引入组件

import { Cell, Checklist } from 'bh-mint-ui';

export default {
    components: {
       Cell,
       Checklist
    }
}
