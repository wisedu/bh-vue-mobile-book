# 移动端开发框架简介

## 基本介绍

开发框架需要承担的功能:

1. 整合前端移动组件库
2. 模拟外框 Hybrid 能力
3. mock 服务端 api
4. 开发环境的运行
5. 上线代码打包

## 开发技术栈

* [vue](http://cn.vuejs.org/)
* [vue-router](http://router.vuejs.org/zh-cn/index.html)
* [vuex](http://vuex.vuejs.org/zh-cn/index.html)
* [OnsenUI](http://vuex.vuejs.org/zh-cn/index.html)
